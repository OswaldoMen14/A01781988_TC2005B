Cheat sheet de los diferentes tags y elementos en HTML.

En este proyecto se elaborará una "cheat sheet", acordeón en español, con algunos de los diferentes tags que existen en HTML, así como su descripción y un ejemplo de su uso.

a - Hyperlinks

Este tag se encarga de definir un hyperlink, hipervínculo en español, el cual hace referencia a algún otro recurso dentro de la misma web. Por ejemplo:

Liga hacia el trailer de Marvel's Spider-Man 2 PS5!

p - Párrafos

Este tag es el encargado de distribuir el texto en párrafos, haciendo que este sea mucho más comprensible para el lector. Por ejemplo, este párrafo y la descripción de los demás tags fue elaborado usando estas etiquetas: 

<p></p>

h1, h2, ... , h6 - Etiquetas de encabezado

Estos tags son utilizados para dar la importancia a los encabezados del texto, los cuales están definidos por jerarquía del 1 al 6, siendo 1 el más importante y así sucesivamente. Ejemplos de encabezados:

Encabezado 1

Encabezado 2

Encabezado 3

Encabezado 4

Encabezado 5
Encabezado 6

div - División

Este tag nos permite crear secciones o agrupar contenidos dentro de una página. Así, se trata de un bloque que se usa para identificar grandes agrupaciones de contenido dentro de la página.

span - Contenedor en línea

Se trata de un tag que crea un contenedor en línea, agrupando contenido que puede ser posteriormente modificado, como por ejemplo,para agregar estilo o color al texto. Ejemplo: Esto es un texto en línea agrupado y con color elaborado con span.

strong - Negritas

Este tag permite resaltar un texto en negritas para hacer énfasis en este, dándole su respectivo nivel de importancia. Por ejemplo: Batman Arkham City es el mejor juego de superhéroes de todos los tiempos.

i - Itálicas

Este tag permite poner un texto en itálicas, también conocida como cursiva, la cual sirve para hacer referencia a alguna frase escrita en algún otro idioma, una cita textual, etc.
Ejemplo: "Un gran poder conlleva una gran responsabilidad" - tío Ben, 2002.

u - Subrayar

Este tag nos permite subrayar algún texto, como por ejemplo, palabras clave o ideas principales y secundarias del texto.


Ejemplo: La guerra de consolas es innecesaria, cada quien debe disfrutar sus juegos sin importar la opinión de terceros.

b - Negritas

Al igual que strong, se encarga de resaltar en negritas un elemento del texto. Sin embargo, strong se debe utilizar cuando el texto tiene una fuerte importancia o un sentido de urgencia, mientras que b se usa solo para llamar la atención sobre un tramo de este texto.
Ejemplo: Kratos le indicó a Atreus que preparara su arco para la batalla.

strike - Elemento tachado

Esta etiqueta sirve para tachar el texto mostrado con una lista horizontal, como por ejemplo, se puede usar para tachar los elementos hechos de una lista. Ejemplo:
Comprar Resident Evil 4 remake

small - Disminuir el tamaño del texto

Esta etiqueta sirve para disminuir el tamaño del texto en uno, por ejemplo, para describir los derechos de autor de un producto.
Ejemplo: El contenido de este juego está protegido bajo la licencia de Sony Interactive Entertainment.

em - Enfasis

Este tag permite marcar con énfasis las partes importantes de un texto. Ejemplo:
Este texto está marcado con énfasis para demostrar su importancia.

header - Encabezado

Sirve para denotar el encabezado de nuestra página. Puede contener el título de la página, el logo, navegación, etc. Ejemplo:

Aquí va el encabezado de la página y sus elementos visuales.

section - Sección

Esta etiqueta sirve para indicar una sección de la página, la cual determina qué contenido corresponde a qué parte dentro de la misma. Ejemplo:

Sección importante de la página

Contenido que va dentro de esta sección.

article - Artículo

Como su nombre lo indica, se trata de un artículo creado dentro de la página, como por ejemplo, un mensaje posteado en un foro, un artículo de revista o científico, entrada de un blog, etc.
Ejemplo:

Reseña de Persona 5 Royal.

¡El mejor JRPG contemporáneo!

aside - Contenido lateral

Esta etiqueta permite crear un contenedor para contenido secundario o que esté relacionado directamente con el contenido principal de la página. Ejemplo:

¡Sigue las redes sociales de PlayStation!

  • Youtube

Esta etiqueta permite crear un pie de página, el cual contiene, como por ejemplo, información relevante respecto a los derechos de autor, logo de la empresa, etc.
Ejemplo:

Este es el pie de página.

nav - Navegación

Este elemento permite proporcionar enlaces de navegación dentro de la página, como por ejemplo:

  • Inicio.
  • Información sobre el videojuego.
  • Contacta con el equipo.

img - Imagen

Como su nombre lo indica, esta etiqueta permite la inserción de una imagen en el documento.

Este texto contendrá un salto de línea después de la siguiente palabra
Y aquí continúa con el final del texto.

form - Interacción con el usuario

Esta etiqueta permite la inserción de un componente en el cual existe una interacción en la que se recolecta información del usuario de la página, como por ejemplo:

Nickname del jugador:

table - Tabla

Como su nombre lo indica, esta etiqueta sirve para insertar una tabla dentro de la página. tr se usa para representar la fila contenedora y td la celda en específico y su contenido.
Ejemplo:

Contenido 1 Contenido 2 Contenido 3
Contenido 4 Contenido 5 Contenido 6

ul - Lista no ordenada

Como su nombre lo indica, esta etiqueta nos permite insertar una lista no ordenada, es decir, una lista en donde los elementos están relacionados entre sí pero sin ningún orden o secuencia establecida.
A continuación se muestra un ejemplo de ello:

  • Resident Evil remake
  • Resident evil 4
  • Resident evil 3:Nemesis

ol - Lista ordenada

Parecida a la anterior, solo que esta vez permite ordenar la lista, ya sea con números o letras.
Ejemplo:

  1. Batman
  2. Superman
  3. Wonder Woman
