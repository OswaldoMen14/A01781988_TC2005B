Preguntas

¿Qué es el DOM?

El DOM (Document Object Model) es una representación en memoria de un documento HTML que permite al programador acceder y manipular los elementos de una página web a través de código JavaScript.

Fuente: Mozilla Developer Network

¿Qué es un nodo?

En el DOM, un nodo es un elemento de un documento HTML que puede ser manipulado mediante código JavaScript. Cada nodo representa una parte del documento, como un elemento, un atributo o un fragmento de texto.

Fuente: Mozilla Developer Network

¿Cómo se accede a un elemento del DOM?

Existen varias formas de acceder a un elemento del DOM mediante código JavaScript, algunas de ellas son:

  • Utilizando el método getElementById(), que busca un elemento por su atributo id.
  • Utilizando el método getElementsByClassName(), que busca elementos por su atributo class.
  • Utilizando el método getElementsByTagName(), que busca elementos por su etiqueta HTML.
  • Utilizando el método querySelector(), que busca un elemento utilizando selectores CSS.

Fuente: Mozilla Developer Network

¿Cómo se modifica un elemento del DOM?

Para modificar un elemento del DOM mediante código JavaScript, se puede utilizar el método setAttribute() para cambiar un atributo del elemento, o los métodos innerHTML y textContent para modificar el contenido del elemento.

Por ejemplo, para cambiar el valor del atributo src de una imagen:

let img = document.getElementById("mi-imagen");
        img.setAttribute("src", "nuevo-valor.jpg");

O para modificar el contenido de un elemento:

let parrafo = document.getElementById("mi-parrafo");
        parrafo.textContent = "Este es el nuevo contenido del párrafo.";

Fuente: Mozilla Developer Network

¿Cuál es la diferencia entre getElementby* y querySelector*?

La diferencia principal entre los métodos que empiezan con getElementBy y los que empiezan con querySelector es que los primeros sólo pueden buscar elementos por su nombre de etiqueta, id o nombre de clase, mientras que los segundos pueden buscar cualquier elemento utilizando selectores CSS.

Por ejemplo, si tenemos el siguiente HTML:

<ul id="mi-lista">
        <li class="item">Item 1</li>
        <li class="item">Item 2</li>
        <li class="item">Item 3</li>
        </ul>

Para acceder al segundo elemento de la lista, podemos usar:

// Usando getElementBy*
        let lista = document.getElementById("mi-lista");
        let elemento = lista.getElementsByTagName("li")[1];

        // Usando querySelector*
        let elemento = document.querySelector("#mi-lista .item:nth-of-type(2)");

Fuente: Mozilla Developer Network

¿Qué es un evento?

En programación web, un evento es cualquier acción que se produce en una página web, como hacer clic en un botón, mover el ratón o pulsar una tecla. Los eventos son utilizados para crear páginas interactivas que respondan a la interacción del usuario.

En JavaScript, se pueden detectar y manejar eventos utilizando el DOM (Modelo de Objetos del Documento). Para hacerlo, se utiliza el método addEventListener(), que permite asignar una función a un evento específico. Por ejemplo, para detectar cuando se hace clic en un botón:

let boton = document.getElementById("mi-boton");
        boton.addEventListener("click", function() {
        console.log("Se ha hecho clic en el botón.");
        });

Existen muchos tipos de eventos diferentes, incluyendo eventos del ratón, eventos de teclado, eventos de carga y descarga, eventos de formulario, eventos de ventana, entre otros.

Fuente: Mozilla Developer Network

¿Qué es un listener?

En programación web, un listener (o "escuchador" en español) es una función que se encarga de "escuchar" un evento específico en un elemento de una página web y ejecutar cierto código cuando ese evento ocurre.

Los listeners se pueden añadir a los elementos del DOM utilizando el método addEventListener() en JavaScript. Por ejemplo, para añadir un listener que se ejecute cuando se hace clic en un botón:

let boton = document.getElementById("mi-boton");
        boton.addEventListener("click", function() {
        console.log("Se ha hecho clic en el botón.");
        });

En este caso, la función que se pasa como segundo parámetro al método addEventListener() es el listener. Cuando se hace clic en el botón, se ejecuta la función, que en este caso sólo muestra un mensaje en la consola del navegador.

Fuente: Mozilla Developer Network

¿Qué es un callback?

En programación, un callback es una función que se pasa como argumento a otra función para que sea ejecutada después de que ésta termine su tarea. El callback se utiliza para manejar los resultados de la tarea asíncrona.

En JavaScript, los callbacks son ampliamente utilizados para manejar tareas asíncronas, como solicitudes a servidores web o animaciones. Por ejemplo, para realizar una solicitud AJAX con jQuery:

$.ajax({
        url: "miarchivo.txt",
        success: function(data) {
            console.log("La solicitud ha sido completada.");
            console.log(data);
        }
        });

En este caso, la función que se pasa como valor del parámetro "success" es el callback. Cuando la solicitud AJAX se completa satisfactoriamente, se ejecuta la función, que en este caso sólo muestra un mensaje en la consola y el contenido del archivo "miarchivo.txt".

Fuente: Mozilla Developer Network

¿Qué es un objeto de evento?

En programación web, un "objeto de evento" es un elemento que contiene información sobre un suceso que ha ocurrido dentro del DOM, como un clic de ratón o una pulsación de tecla. El objeto de evento se crea automáticamente por el navegador y se pasa como argumento a cualquier listener o función que se haya registrado para manejar el evento.

Asimismo, tiene propiedades que describen el tipo de evento, el elemento del DOM en el que ocurrió, las coordenadas del ratón en el momento del evento, entre otras. También puede tener métodos que permiten detener la propagación del evento o prevenir su comportamiento predeterminado.

En JavaScript, se puede acceder al objeto de evento dentro de un listener o función utilizando el parámetro "event". Por ejemplo, para mostrar un mensaje que incluya las coordenadas del ratón cuando se hace clic en un botón:

let boton = document.getElementById("mi-boton");
        boton.addEventListener("click", function(event) {
        console.log("Se ha hecho clic en el botón en la posición (" + event.clientX + ", " + event.clientY + ").");
        });

En este caso, el objeto de evento se utiliza para acceder a las propiedades "clientX" y "clientY", que contienen las coordenadas del ratón en el momento del clic.

Fuente: Mozilla Developer Network
